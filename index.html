The 'font-family' Property:

Google Fonts, Common Fonts and Websafe Fonts

When choosing a font for your website and listing it in the font-family property in your CSS style sheet, it is a good idea to list three font families so different browsers can display the content of the page should your preferred font not be available.

Google Fonts

There are many fonts hosted by Google ready to be used in a webpage at: https://fonts.google.com.

Here are two ways to do this:

  1. To embed a font, copy the code into the <head> of your html.
  2. In the CSS style sheet, use the @font-face to download a custom font with which to display text; the font can be loaded from either a remote server or a locally-installed font on the user's own computer.
